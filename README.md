## 🔥 Featured Projects
 
### 🤖 [AI Study Assistant — RAG-Powered PDF Chatbot](https://github.com/Dhayanidhi-96/ai-study-assistant)
> **Live:** [huggingface.co/spaces/Dhayanidhi-96/ai-study-assistant](https://huggingface.co/spaces/Dhayanidhi-96/ai-study-assistant)
 
A production-ready conversational AI that lets students upload any PDF and ask questions — powered by **Retrieval-Augmented Generation (RAG)**.
 
**Tech Stack:**
`Python` `LangChain` `LLaMA 3.1` `Groq API` `FAISS` `HuggingFace Embeddings` `Streamlit` `Docker`
 
**Key Highlights:**
- History-aware retriever using LangChain for coherent multi-turn conversations
- Semantic search over FAISS vector store with HuggingFace sentence embeddings
- Near-instant responses via Groq's LPU-accelerated LLaMA 3.1 inference
- Containerized with Docker and deployed on Hugging Face Spaces
- Launch post reached **10,000+ LinkedIn impressions** and **6,900+ members reached** in 24 hours

A real-time emotion recognition system using a custom CNN trained on FER-2013, with live webcam inference via OpenCV.
 
**Tech Stack:**
`Python` `TensorFlow` `Keras` `OpenCV` `CNN` `FER-2013`
 
**Key Highlights:**
- Custom CNN trained on 35,887 images across 7 emotion classes
- 68% validation accuracy with data augmentation (rotation, flipping, zoom)
